# classical-NP-complete-problem
## [Лабораторная] Классическая NP-полная задача

В этой лабораторной работе Вам предлагается решить задачу коммивояжера. Однако, помимо рассмотренных ранее методов, требуется подойти к данной комбинаторной задаче с точки зрения вероятностей: муравьиным алгоритмом и методом отжига (simulated annealing). Также их предлагается сравнить с методом остовного дерева (minimal spanning tree).

Для решения задачи, таким образом, необходимо:

1. Реализовать алгоритм генерации графа с выбранной плотностью (от разреженного до плотного) (закомментированный код)
2. Реализовать упомянутые выше алгоритмы обхода графа
3. Сгенерировать n графов, увеличивая их плотность
4. Проверить на этих графах каждый из алгоритмов
5. Сравнить результаты (в идеале - визуализировать). Выяснить, для графов какой плотности какой алгоритм эффективнее и почему
